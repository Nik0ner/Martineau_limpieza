# Sitio Web – Limpieza Profesional de Alfombras

Sitio web corporativo desarrollado para una empresa de **limpieza de alfombras, cortinas y tapices**.  
El proyecto está enfocado en una **landing page moderna**, clara y orientada a la **conversión de clientes**.

---

## Descripción del Proyecto

Este sitio web presenta los servicios de limpieza de forma clara y profesional, utilizando una estructura de **single-page (one page)** con navegación por secciones.

El objetivo principal es:
- Mostrar los servicios
- Generar confianza en el cliente
- Facilitar el contacto y cotización

---

- **CSS3**
- **Bootstrap 5**
